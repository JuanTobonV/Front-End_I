Manipulación de datos con LocalStorage

LocalStorage es una herramienta compatible con html y los navegadores, gracias a esta herrmienta podemos almacenar de manera local datos en el navegador que quedará almacenada para que siempre qu el usuario inicie en su navegador estos datos esten ahí. LocalStorage trabaja de manera similar con las cookies, sin embargo, cuenta con un espacio de 5mb y al trabajar de manera local no necesita hacer peticiones al servidor.

¿Cómo se utiliza LocalStorage

LocalStorage es una herramienta que se utiliza via Javascript. Para acceder a este almacenamiento se hace: Una vez ejecutada la aplicación 1. Inspeccionar 2. apartado de application 3. en el menu izquierdo aparece el LocalStorage

Metodos del LocalStorage

1. LocalStorage.setItem("clave", valor) -> este es el metodo para almacenar información en nuestro LocalStorage. En el primer parametro "clave" es donde ingresamos el nombre del elemento y en el valor va el dato que almacenará el parametro clave

localStorage.getItem("clave") -> Este metodo me sirve para traer el valor almacenado en una clave especifica

Intercambio de datos por medio de JSON

Para poder almacenar arreglos y objetos en el localStorage se debe de convertir en un string, para ello usaremos JSON
JSON.stringify()

Convertir un String almacenado en el localStorage en un objeto: JSON.parse(string)
